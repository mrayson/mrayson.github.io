Ocean Modelling

I have developed several tools for the pre/post-processing of numerical model and oceanographic measurment data. The majority of these are written in python and are hosted on the SUNTANS SourceForge page. This includes:

  • Python toolbox for the SUNTANS unstructured grid model.
  • 3D unstructured grid visualisation in python using Mayavi.
  • MATLAB and python tools for setting up Regional Ocean Modeling System (ROMS) data
    • See my ROMS python module.
